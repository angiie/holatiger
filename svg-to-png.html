SVG 转 PNG 在线工具

SVG 是矢量图（任何尺寸不失真），但很多场景——PPT、PDF、社交平台头像、App 图标——需要 PNG 位图。SVG Studio 在浏览器内把 SVG 渲染为 Canvas 后导出 PNG，文件不上传服务器。

常见问题

SVG 转 PNG 会有质量损失吗？

SVG 是矢量图，转 PNG 后变成位图。从 100x100 放大到 1000x1000 会变模糊。建议按目标尺寸直接导出 1x、2x、3x 三种密度（SVG Studio 支持）。

支持批量转换吗？

支持。一次拖入多个 SVG，逐个渲染并打包为 ZIP。

输出 PNG 的最大尺寸是多少？

理论上无限制。实际受浏览器 Canvas 限制（通常 16384 x 16384），Chrome 扩展图标包场景下用不到这么大。

能保留透明背景吗？

可以。默认透明背景。如果需要白底，可启用「填充背景色」。
