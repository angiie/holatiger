# SVG Studio Pro

一个专业、免费、高效的 SVG 查看器和导出工具，专为设计师、开发者和任何需要处理 SVG 文件的人打造。它完全在你的浏览器中运行，无需安装任何软件，也无需上传文件到服务器，确保你的数据安全。

## ✨ 功能特性

*   **实时 SVG 预览**:
    *   **多种导入方式**: 支持拖拽 SVG 文件、从本地选择文件或直接粘贴 SVG 源代码。
    *   **即时渲染**: 在你输入或修改 SVG 代码时，预览区域会立即更新。

*   **灵活的导出选项**:
    *   **多种格式**: 支持将 SVG 文件导出为 PNG、ICO、WebP 和 JPG 格式。
    *   **自定义尺寸**: 可以自由设置导出图像的宽度和高度。
    *   **预设尺寸**: 提供了一系列常用的预设尺寸，如 16x16, 32x32, 128x128 等，方便你快速选择。

*   **强大的批量处理**:
    *   **批量导出**: 可以选择多个尺寸，一次性将 SVG 导出为包含所有选定尺寸的 PNG 图片的 ZIP 压缩包。
    *   **Chrome 图标包**: 一键生成符合 Chrome 网上应用店要求的完整图标包（manifest.json 和所有尺寸的图标）。

*   **用户友好的界面**:
    *   **多语言支持**: 支持简体中文、繁體中文和英文。
    *   **直观易用**: 简洁的界面设计，让你无需学习即可上手。
    *   **轻量高效**: 完全基于 HTML, CSS 和原生 JavaScript 构建，加载速度快，性能卓越。

## 🚀 如何使用

1.  **打开页面**: 在浏览器中打开 `index.html` 文件。
2.  **导入 SVG**:
    *   将你的 `.svg` 文件拖拽到上传区域。
    *   点击“选择文件”按钮，从你的电脑中选择一个 `.svg` 文件。
    *   将 SVG 代码直接粘贴到“SVG 代码”文本框中。
3.  **预览和调整**:
    *   在右侧的预览区域查看你的 SVG 图像。
    *   在“尺寸设置与导出”选项卡中，输入你想要的宽度和高度，或选择一个预设尺寸。
4.  **导出**:
    *   **单个导出**: 点击“下载 PNG”或“下载 ICO”按钮。
    *   **批量导出**:
        *   切换到“批量导出”选项卡。
        *   选择你需要的尺寸。
        *   点击“批量导出 PNG”或“Chrome 图标包”按钮。

## 🛠️ 技术栈

*   **前端**: HTML, CSS, JavaScript
*   **第三方库**: [JSZip](https://stuk.github.io/jszip/) 用于生成 ZIP 文件。
