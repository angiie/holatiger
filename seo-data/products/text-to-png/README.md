# Text to PNG — SEO Strategy

## 产品定位
浏览器内把文字渲染为 PNG 图片的工具，主打**自定义字体、颜色、阴影、描边**。

## 核心关键词
- 主词：text to png / text to image / create text image
- 长尾：text to png with custom font / text png online / font png maker
- 用途：banner / social / placeholder

## 目标竞品
1. canva.com（DR 92+，多功能设计）
2. figma.com（DR 90+，设计）
3. cooltext.com（DR 50+，logo 文字）
4. textcraft.net（DR 30+，Minecraft 风格文字）
5. flamingtext.com（DR 55+，logo 文字）

## 链接建设策略
- **类型 1：营销/社交媒体博客**（如 buffer、hootsuite 博客）
- **类型 2：字体设计社区**（如 myfonts、dafont 论坛）
- **类型 3：PPT/Keynote 教程站**（如 presentitude、24slides）
