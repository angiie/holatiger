# WebDavy — SEO Strategy

## 产品定位
Chrome 扩展，把 WebDAV 云盘（Nextcloud / 坚果云 / 群晖 / ownCloud）接入浏览器。

## 核心关键词
- 主词：webdav chrome / webdav client / webdav extension
- 长尾：nextcloud chrome / 坚果云 chrome / 群晖 webdav / owncloud browser
- 竞品对比：webdavy vs raidrive / webdavy vs cyberduck

## 目标竞品
1. raidrive.com（Windows 磁盘符挂载）
2. cyberduck.io（Mac/Win 通用 FTP/WebDAV 客户端）
3. mountainduck.io（同上）
4. odrive.com（多云聚合）
5. insync.app（Google Drive 同步）

## 链接建设策略
- **类型 1：自托管社区客座**（如 awesome-selfhosted、Reddit r/selfhosted）
- **类型 2：NAS 论坛评测**（NASCompares、chiphell、什么值得买）
- **类型 3：开发者工具聚合**（如 Chrome 扩展评测站）
