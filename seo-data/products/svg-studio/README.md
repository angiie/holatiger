# SVG Studio — SEO Strategy

## 产品定位
浏览器内 SVG 转换/导出工具，主打**多格式输出、Chrome 图标包、批量**。

## 核心关键词
- 主词：svg to png / svg converter / svg editor
- 长尾：svg to ico / svg to webp / chrome extension icon generator
- 竞品对比：svg studio vs inkscape / svg studio vs svgomg

## 目标竞品
1. cloudconvert.com（DR 80+，多功能在线转换）
2. svgomg.com（DR 30+，SVG 优化器）
3. convertio.co（DR 75+，多功能）
4. inkscape.org（DR 78+，桌面矢量编辑器）
5. figma.com（DR 90+，设计工具）

## 链接建设策略
- **类型 1：Chrome 扩展开发者社区**（如 stackoverflow、github awesome-*）
- **类型 2：设计师博客客座**（如 smashingmagazine、css-tricks）
- **类型 3：矢量素材网站**（如 svgsilh、iconify）
