# TinyPass — SEO Strategy

## 产品定位
本地运行的强密码生成器，主打**可读模式、批量、零注册**。

## 核心关键词
- 主词：password generator / random password / strong password
- 长尾：password generator readable / batch password generator / password generator local
- 竞品对比：tinypass vs 1password / tinypass vs lastpass / tinypass vs bitwarden

## 目标竞品
1. lastpass.com（密码管理器，含生成器）
2. 1password.com（密码管理器）
3. bitwarden.com（开源密码管理器）
4. randomkeygen.com（纯生成器）
5. dinopass.com（弱密码生成器 - 低质竞品，不直接对标）

## 链接建设策略
- **类型 1：安全/隐私博客客座**（如 securityweek、krebsonsecurity）
- **类型 2：开发工具聚合**（如 producthunt、saashub）
- **类型 3：Reddit r/privacy、r/netsec 分享**
