# Tiny Pic — SEO Strategy

## 产品定位
浏览器内 WASM 加速的图片压缩工具，主打**本地处理、批量、零注册**。

## 核心关键词
- 主词：image compressor / compress jpeg / jpg compressor
- 长尾：batch image compressor / local image compressor / compress image without losing quality / wasm image compressor
- 竞品对比：tinypic vs squoosh / tinypic vs tinypng / tinypic vs compressor.io

## 目标竞品（3-5）
1. tinypng.com（DR 80+，传统 web 服务）
2. squoosh.app（Google Chrome Labs，DR 70+）
3. compressor.io（DR 50+）
4. shortpixel.com（DR 60+）
5. iloveimg.com（DR 70+，多功能）

## 链接建设策略
- **类型 1：技术博客客座**（高 DR 编程/设计博客，锚文本用品牌名 + 主词）
- **类型 2：工具聚合页收录**（如 producthunt.com/alternatives、saashub.com）
- **类型 3：本地化目录**（如国内的 tools.cat/tools.wf、国外的 awesome-selfhosted）
