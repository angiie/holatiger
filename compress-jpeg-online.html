在线压缩 JPEG 图片

JPEG 是互联网上最常见的图片格式之一，但单张照片动辄几 MB，拖慢页面加载、占用云盘空间。Tiny Pic 是一个完全本地运行的 JPEG 压缩工具，使用 WebAssembly 加速的 libjpeg-turbo 编码器，文件不离开你的设备。

为什么选择本地压缩

市面上多数在线压缩工具会先把你的图片上传到服务器，处理后再下载回浏览器。这个过程有三重隐患：

  • 隐私风险：个人照片、合同截图、产品图都可能经过第三方服务器。
  • 速度限制：上传耗时受网络带宽限制，10 MB 的图可能 30 秒。
  • 使用限制：多数工具限制免费次数或单图大小。

Tiny Pic 把 libjpeg-turbo 编译成 WebAssembly 跑在浏览器里，从读取到编码全程在你的设备上完成。

常见问题

压缩 JPEG 会损失画质吗？

Tiny Pic 使用 libjpeg-turbo 重新编码 JPEG 文件。质量档位 80+ 几乎无肉眼可感知的画质损失，文件大小可减少 50%-80%。

图片会上传到服务器吗？

不会。Tiny Pic 完全在浏览器内运行，图片通过 File API 读取后在 WebAssembly 中处理，全程不经过任何服务器。

支持批量压缩吗？

支持。你可以一次拖入多张图片，Tiny Pic 会逐个处理并打包为 ZIP 下载。

压缩后的图会保留 EXIF 信息吗？

默认保留。你也可以选择去除 EXIF 信息以进一步缩小文件体积。

最大能压缩多大的图？

理论上没有大小限制。处理超大图片时，浏览器内存是瓶颈。建议单图控制在 50 MB 以内以获得最佳体验。
