WebDAV Chrome 扩展设置指南

WebDavy 把你的 WebDAV 云盘（Nextcloud、坚果云、群晖、ownCloud 等）直接接入 Chrome 浏览器，不用每次打开 Web 客户端。5 分钟完成设置，文件预览、上传、下载、目录书签一步到位。

设置步骤

第 1 步：安装 WebDavy 扩展

在 Chrome / Edge / Brave 等 Chromium 内核浏览器中打开 Chrome Web Store，搜索 "WebDavy" 或直接访问 WebDavy 产品页。点击「添加至 Chrome」。

第 2 步：打开扩展弹窗

安装完成后，工具栏会出现 WebDavy 图标（默认在拼图图标里）。点击图标打开弹窗。

第 3 步：添加账户

点「添加账户」，填写：

  • 账户名称：起个易识别的名字，比如「公司 Nextcloud」
  • 服务器地址：见下表
  • 用户名：你的 WebDAV 账号
  • 密码：建议使用「应用专用密码」

第 4 步：测试连接

点击「测试连接」按钮。如果配置正确，会显示根目录下的文件列表。

第 5 步：添加目录书签

把 /Documents、/Photos 等常用目录添加到书签。下次打开弹窗一键直达。

常见 WebDAV 服务器地址

Nextcloud：https://你的域名/remote.php/dav
自托管用户把"你的域名"换成实际地址。
坚果云：https://dav.jianguoyun.com/dav
用户名 = 邮箱前缀，密码 = 应用专用密码（在坚果云账户安全里生成）。
群晖 Synology NAS：https://你的NAS地址:5005/webdav
需先在 DSM 控制面板 → WebDAV Server 启用。
ownCloud：https://你的域名/remote.php/webdav

常见问题

WebDAV 服务器地址怎么填？

Nextcloud 默认是 https://你的域名/remote.php/dav，坚果云是 https://dav.jianguoyun.com/dav，群晖是 https://你的NAS地址:5005/webdav，ownCloud 通常是 https://你的域名/remote.php/webdav。

WebDavy 是免费的吗？

是。基础功能（多账户、文件预览、上传下载、目录书签）完全免费。

后台常驻会占很多内存吗？

不会。WebDavy 平时是休眠的 Service Worker，按需激活，内存占用通常在 10 MB 以内。

支持多账户吗？

支持。你可以在 WebDavy 中保存多个 WebDAV 账户（个人 Nextcloud + 公司坚果云等），随时切换。
