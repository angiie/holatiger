SVG Studio vs Inkscape

这俩不是同维度的工具：SVG Studio是浏览器内 SVG 转换/导出工具（SVG → PNG/ICO/WebP/JPG），Inkscape是开源桌面矢量编辑器（画 SVG）。选哪个，取决于你要"导出"还是要"创作"。

选哪个

已有 SVG，需要转 PNG/ICO/WebP/JPG（尤其批量） → 选 SVG Studio。零安装，批量 + ZIP。
需要从零画一个新 SVG → 选 Inkscape。专业矢量编辑。
需要编辑现有 SVG 的节点/路径 → 选 Inkscape。SVG Studio 不做节点编辑。

常见问题

SVG Studio 和 Inkscape 是一类产品吗？

不是。SVG Studio 专注转换/导出，Inkscape 是完整的矢量编辑器。需要画新图选 Inkscape；要把已有 SVG 转 PNG/ICO/WebP 选 SVG Studio。

Inkscape 是免费的吗？

是。Inkscape 完全免费开源（GPL）。SVG Studio 也完全免费。
